Pass generics types to an actual object. For instance if we are using the `List` class, we define it like `List<String>` or whatever type we like. These are where we define the types. Can also pass wildcard types like `List<? extends String>`, which allows you to pass unknown types.

<Code code={`import { passGenerics } from "@alloy-js/core";

function passGenerics(generics: (Child | Record<"?", Children>)[]): Child[];`} lang="ts" />
