Shorthand to instantiate a new object. Declares it with 'new' and passes arguments to the constructor of the object, if any

    {children}
</ObjectDeclaration>`} lang="tsx" />
</TabItem>

ObjectDeclaration({ arguments: Children, type: Children}).children(children)` } lang="ts" />
</TabItem>
</Tabs>
