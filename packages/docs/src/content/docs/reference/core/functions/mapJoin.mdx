## Overload 1

Map a Map to an array using a mapper and place a joiner between each element. Defaults to joining with a newline.

<Code code={`import { mapJoin } from "@alloy-js/core";

function mapJoin<T, U, V>(src: Map<T, U>, cb: (key: T, value: U) => V, options?: JoinOptions): (V | string)[];`} lang="ts" />

### Parameters

<table>
  <tr>
    <td style="text-align: right;font-weight: bold;">src</td>
    <td>
    
      `Map<T, U>`
      
      
    </td>
  </tr>
  <tr>
    <td style="text-align: right;font-weight: bold;">cb</td>
    <td>
    
      `(key: T, value: U) => V`
      
      
    </td>
  </tr>
  <tr>
    <td style="text-align: right;font-weight: bold;">options</td>
    <td>
    
      `JoinOptions`
      
      
    </td>
  </tr>
</table>

### Options

<table>
  <tr>
    <td style="text-align: right;font-weight: bold;">ender</td>
    <td>
      <Badge text="optional" variant="note" size="small" /> <code>string | boolean</code>
  
  When true, the joiner is placed at the end of the array. When a string, that string is placed at the end of the array. The ender is only emitted when the array has at least one element.
  
  
  </td>
  </tr>
  <tr>
    <td style="text-align: right;font-weight: bold;">joiner</td>
    <td>
      <Badge text="optional" variant="note" size="small" /> <code>string</code>
  
  The string to place between each element.
  
  
  </td>
  </tr>
</table>

### Returns

<code>(V | string)[]</code>

The mapped and joined array.







### See also


* <a href="/reference/core/functions/join/">join</a> for joining without mapping.
